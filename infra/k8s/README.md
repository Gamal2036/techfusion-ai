# Kubernetes Manifests

This directory will contain Helm charts and Kubernetes manifests starting from **Phase 14**.

The infrastructure here will include:

- Helm chart definitions for all microservices
- Ingress controller configuration
- Service mesh (Istio/Linkerd) setup
- Persistent volume claims
- ConfigMaps and Secrets management
- Horizontal Pod Autoscalers
- Network policies

Phase 14 will replace the Docker Compose setup for production deployments.
